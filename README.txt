##Intrucciones##
Para poder jugar se necesita seguir los siguientes pasos:
1. El primer jugador debe ingresar la letra "C" y presionar enter.
2. El segundo jugador hace el mismo procedimiento.
3. Se repite este procedimiento hasta que uno de los dos jugadores llegue a la casilla 30 o 
   decidan acabar antes presionando la tecla "E".

##Instruccions##
To be able to play this game you need to follow this steps:
1. Player 1 must input the letter "C" and press the enter key.
2. The player 2 does the same thing.
3. Repeat this process until ether one gets to the box number 30 or you decide to end the game early by
   pressing the "E" key.
